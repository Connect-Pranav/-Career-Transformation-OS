## How to Deploy on GitHub Pages

### Step 1 — Create repository
```bash
# Create a new repo named: orion-ai-academy
# Go to: https://github.com/new
# Repository name: orion-ai-academy
# Visibility: Public
```

### Step 2 — Upload files
```bash
git clone https://github.com/connect-pranav/orion-ai-academy.git
cd orion-ai-academy

# Copy all files from this folder into the repo
cp -r /path/to/pranav-ai-academy/* .

git add .
git commit -m "Initial commit: ORION AI Academy"
git push origin main
```

### Step 3 — Enable GitHub Pages
1. Go to your repo → **Settings** → **Pages**
2. Under **Source**, select `main` branch and `/ (root)` folder
3. Click **Save**
4. Wait 1–2 minutes
5. Your site will be live at: `https://connect-pranav.github.io/orion-ai-academy/`

## Features

- **Interactive Tracker** — Check off tasks week by week. Progress auto-saves to `localStorage`. No backend required.
- **Course Filter** — Filter courses by month/phase on the Courses page.
- **Counter Animations** — Stats animate on scroll.
- **Fully Responsive** — Works on mobile, tablet, desktop.
- **No dependencies to install** — Pure HTML/CSS/JS. Zero build step.
