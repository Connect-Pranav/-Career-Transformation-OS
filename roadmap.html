Month-by-Month Detail

What to Learn, Build & Achieve

Month 1 · Jul–Aug 2026
🐍 Python for Data Science & APIs
Foundation sprint. 1–2 hours/day. No prior coding experience required.
📖 Learn
  • Python syntax, loops, functions
  • Pandas DataFrames & Series
  • NumPy for numerical ops
  • REST API calls with Requests
  • JSON parsing & manipulation
🛠️ Build
  • Sales data CSV analyser
  • Simple REST API caller
  • Pandas revenue dashboard
  • Push all to GitHub
📜 Certify
  • IBM Python for Data Science
  • Google AI Essentials (parallel)
  • Generative AI for Everyone
🎯 Month 1 Outcome: Can write Python scripts, call APIs, and manipulate data with Pandas. IBM certificate + Google AI Essentials certificate earned.
Month 2 · Aug–Sep 2026
📊 Analytics: Power BI, SQL & Looker
Deepen your existing analytics muscle with certification-grade expertise.
📖 Learn
  • Power BI DAX measures
  • Data modelling (star schema)
  • SQL window functions
  • Looker dashboards & LookML
  • Report design principles
🛠️ Build
  • magicpin territory Power BI report
  • SQL revenue analysis queries
  • Looker Explore dashboard
  • Publish screenshots to LinkedIn
📜 Certify
  • Microsoft Power BI Data Analyst (PL-300 prep)
  • Google Cloud Data Analytics
  • Salesforce Trailmix badges
🎯 Month 2 Outcome: PL-300-ready. Published 2 Power BI dashboards. SQL intermediate skills. LinkedIn starts generating recruiter interest.
Month 3 · Sep–Oct 2026
🤖 Claude API & Prompt Engineering
The most important month. Directly builds toward AI PM and AI Program Manager roles.
📖 Learn
  • Anthropic API fundamentals
  • XML prompt structuring
  • System prompts & personas
  • Tool use & function calling
  • Token management & costs
🛠️ Build
  • AI Sales Coach chatbot
  • RevOps KPI Analyser
  • AI ROI business case builder
  • SOP Generator with Claude
📜 Certify
  • Building with Claude API (Coursera)
  • Anthropic Academy Certificate (free)
  • DeepLearning.AI Prompt Engineering
🎯 Month 3 Outcome: 3 Claude API projects live on GitHub. Official Anthropic certificate. Can demo AI tools to hiring managers confidently.
Month 4 · Oct–Nov 2026
🔍 RAG Systems & LangChain
Learn to build systems that give AI access to your own data — the core of most enterprise AI products.
📖 Learn
  • Embeddings & vector stores
  • Semantic vs keyword search
  • LangChain document loaders
  • Chunking strategies
  • Hybrid retrieval & reranking
🛠️ Build
  • RevOps Knowledge Base RAG
  • Contract Intelligence System
  • Sales Playbook Q&A bot
  • Deploy on Streamlit / HuggingFace
📜 Certify
  • RAG (DeepLearning.AI, Coursera)
  • LangChain: Chat with Your Data (free)
  • Knowledge Graphs for RAG (free)
🎯 Month 4 Outcome: 2 RAG apps live. Can explain embeddings, vector stores, and retrieval to non-technical stakeholders. Portfolio now interview-worthy.
Month 5 · Nov–Dec 2026
🦾 AI Agents: LangGraph, CrewAI & AutoGen
The cutting edge. Multi-agent systems are what separate AI builders from AI users.
📖 Learn
  • ReAct agent pattern
  • LangGraph state machines
  • CrewAI role-based agents
  • Tool definition & calling
  • Agent memory & persistence
🛠️ Build
  • NEXUS Sales Coach Agent
  • Multi-Agent RevOps Crew
  • AI SDR + Pipeline Analyst Agent
  • Slack-integrated nightly digest bot
📜 Certify
  • Agentic AI with LangGraph, CrewAI, AutoGen (IBM Coursera)
  • AI Agents in LangGraph (free DL.AI)
  • Multi AI Agent Systems with crewAI (free)
🎯 Month 5 Outcome: 3+ agent projects live. IBM Agentic AI certificate. GitHub has 10+ repos. Begin applying to Razorpay/CRED/Urban Company.
Month 6 · Dec 2026–Jan 2027
🚀 NEXUS OS Capstone + Job Hunt
Integrate everything. Build the flagship portfolio project. Convert interviews to offers.
📖 Learn
  • LLMOps basics (LangSmith, MLflow)
  • AWS AI services overview
  • AI PM interview prep
  • AI ROI frameworks for execs
  • STAR stories for AI projects
🛠️ Build
  • NEXUS Business OS v2 (agents + RAG + dashboard)
  • AI career portfolio site
  • 2 AI PRD documents
  • IIM Kozhikode capstone with AI theme
📜 Certify + Apply
  • AWS Cloud Practitioner Essentials
  • AI PM Specialization (Duke)
  • Apply to 5 target companies
  • Target: offer by Jan 20, 2027
🎯 Month 6 Outcome: NEXUS OS live. 12+ certifications. 15+ GitHub repos. Offer from target company at ₹22–26L minimum. Salary floor: ₹18L.
